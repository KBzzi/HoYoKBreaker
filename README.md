这个可以使你运行未启动驱动保护的原神

目前只支持4.2.0 国服、国际服

如果你有什么疑惑或者程序出现BUG，请报告问题

Currently, only 4.2.0 Chinese version and global version are supported

If you have any doubts or program bugs, please report the issue
