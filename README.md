## 使用方法How to use

1. 确保以下文件在HoYoKBreak.exe的根目录下

2. Bak文件夹、config.ini

3. 在config.ini中的：[setting]--GamePath里填入你的游戏的位置，例如：GamePath = D:\Genshin Impact Game

4. 双击打开HoYoKBreak.exe

5. Enjoy game
